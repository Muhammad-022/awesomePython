# Day 66: SQLite Basics

## Concept
SQLite is a lightweight database built into Python.

Python provides the `sqlite3` module to work with SQLite databases.

## What is SQLite?
- File-based database (no server needed)
- Stores data in tables (rows & columns)
- Used in apps, small systems, and testing

## sqlite3 Module

### Import
import sqlite3

## Steps to Use SQLite

1. Connect to database  
   - Creates file if not exists  

2. Create cursor  
   - Used to execute SQL commands  

3. Create table  
   - Define structure  

4. Insert data  
   - Add records  

5. Fetch data  
   - Retrieve records  

6. Close connection  

## SQL Basics Used

- CREATE TABLE → create table  
- INSERT INTO → add data  
- SELECT → fetch data  

## Use Case
- Store user data
- Manage applications data
- Replace CSV/JSON for structured data

## Why Use SQLite?
- Lightweight
- No installation needed
- Fast and simple database
