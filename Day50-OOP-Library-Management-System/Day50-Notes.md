# Day 50: OOP Mini Project (Library Management System)

This project builds a Library Management System using Object-Oriented Programming (OOP).
It manages books, issuing, returning, and searching operations.

### OOP Concepts Used
* Classes
* Objects
* Constructor (init)
* Attributes
* Methods
* Encapsulation
* Object interaction
* Class Design
* Book Class
Represents a single book.
### Attributes:
* title
* author
* book_id
* is_issued
### Features
* Add new books
* View all books
* Issue book
* Return book
* Search book
