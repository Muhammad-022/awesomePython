# Day 63: Mini DSA Project: To-Do List App

## Concept
A To-Do List App helps manage daily tasks efficiently.

This project uses:
- Lists to store tasks
- Stack/Queue concepts to manage task flow

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## Data Structure Used

### List
Used to store tasks.

Example:
["Study Python", "Exercise", "Read book"]

### Stack Concept
- Last added task can be handled first (LIFO)

### Queue Concept
- Tasks handled in order (FIFO)

## How It Works

1. Add Task  
   - Task is added to the list  

2. View Tasks  
   - Display all tasks with index  

3. Complete Task  
   - Mark task as completed  

4. Delete Task  
   - Remove task from list  

## Why This Project?
- Real-world application
- Practice of lists and logic building
- Foundation for advanced apps (GUI/Web)
